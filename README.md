# Number Analyzer

##  Description
This is a Python program that:
- Counts total numbers entered by user
- Calculates sum
- Finds even numbers count
- Supports decimal values
- Stops when user types "stop"

## Technologies Used
- Python

##  How to Run
1. Open the file in Python
2. Run the program
3. Enter numbers
4. Type "stop" to end

##  Features
- Accepts integers and decimal numbers
- Handles invalid input
- Rejects empty input
- Displays result clearly
